# Get variable labels

**\[deprecated\]**

## Usage

``` r
get_var_labels(datasets, dataname, vars)
```

## Arguments

- datasets:

  (`teal::FilteredData`)  
  Data built up by teal

- dataname:

  (`character`)  
  name of the dataset

- vars:

  (`character`)  
  Column names in the data

## Value

`character` variable labels.
